# Training Data

Store your GTZAN dataset and audio files here.

## Structure

```
data/
├── raw/              # Original unprocessed audio files
├── processed/        # Preprocessed audio and spectrograms
└── splits/          # Train/val/test splits
```

## Dataset Info

The GTZAN Music Genre Classification Dataset contains:

- 1000 audio tracks
- 10 genres (100 tracks each)
- 30 seconds per track
- 22050 Hz sample rate

### Genres

1. blues
2. classical
3. country
4. disco
5. hiphop
6. jazz
7. metal
8. pop
9. reggae
10. rock
